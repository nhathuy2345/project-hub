# IT Project Hub

Trang web gioi thieu va tu van cac du an website tot nghiep cho sinh vien IT.

## Chuc nang

- Hien thi danh sach du an theo nhom cong nghe.
- Loc du an theo Laravel, React, ASP.NET va Bao cao luan.
- Form gui yeu cau tu van.
- Luu thong tin lien he vao MySQL bang PHP.
- Trang admin xem danh sach yeu cau tu van.
- Nut lien he nhanh qua Zalo.

## Chay voi XAMPP

1. Copy project vao `C:\xampp\htdocs\it-project-hub`.
2. Bat Apache va MySQL trong XAMPP.
3. Vao `http://localhost/phpmyadmin`.
4. Import file `database.sql`.
5. Mo website tai `http://localhost/it-project-hub/index.html`.
6. Xem du lieu form tai `http://localhost/it-project-hub/admin_contacts.php`.

## Luu y khi deploy

GitHub Pages chi chay duoc HTML, CSS va JavaScript tinh. Cac file PHP va MySQL can hosting co ho tro PHP/MySQL neu muon form luu du lieu that tren internet.
